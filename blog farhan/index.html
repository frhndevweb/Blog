MY WEBSITE DEVELOPMENT JOURNEY

Welcome to the creation Blog Farhan Dev Website

Stage 1: Planning & HTML Setup

Oktober 10, 2024

My journey began with planning the structure of my website. I created a basic `index.html` to lay out sections like the header, main content, and footer. This step focused on creating a clean HTML skeleton that would hold all content.

Stage 2: Styling with CSS

Oktober 12, 2024

Next, I added CSS to style each section of the site. This included designing a responsive layout, choosing colors, and setting typography to match my personal style. I focused on creating a cohesive look for my portfolio.

Stage 3: Adding JavaScript for Interactivity

Oktober 15, 2024

I then added JavaScript to make the site more dynamic. JavaScript helped add interactivity, like hover effects and button actions, which made the website feel more engaging.

Stage 4: Testing Responsiveness

Oktober 18, 2024

Ensuring the site looked great on all devices was crucial. I tested on various screen sizes and adjusted CSS as needed to make sure it was fully responsive, especially for mobile users.

Stage 5: Deploying on Netlify

Oktober 31, 2024

The final step was deploying my site. I chose Netlify for its ease of use. After uploading my files directly to Netlify, my website was live at https://frhndevweb.netlify.app.
